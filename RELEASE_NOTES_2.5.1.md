# 🎉 版本 2.5.1 发布说明

## 📅 发布日期：2025-03-13

---

## ✨ 主要更新

### 🐛 关键修复

1. **视频解析失败** - 修复了无法获取音轨信息的问题
2. **音频提取失败** - 修复了 FFmpeg 退出码 null 的问题
3. **中文文件名支持** - 现在可以正确处理包含中文的文件名
4. **路径处理** - 修复了输出目录初始化失败的问题

### 🆕 新功能

1. **拖拽替换提示** - 拖动新视频时会询问是否替换当前视频
2. **点击选择视频** - 占位符现在可以点击，支持文件选择对话框

### 🔄 架构升级

- 迁移到 `ffmpeg_kit_flutter_new 2.0.0`（活跃维护）
- 应用现在内置 FFmpeg 库（约 99MB）
- 完全独立，不依赖外部 FFmpeg 安装

## 💡 使用提示

### 加载视频

现在有两种方式：
1. **拖拽** - 直接拖拽视频文件到窗口
2. **点击** - 点击占位符区域或"选择文件"按钮

### 替换视频

- 拖动新视频时会提示："是否要替换当前视频？"
- 点击"取消"保持当前视频
- 点击"替换"加载新视频

### 支持的文件

- ✅ 中文文件名：`塞尔达-过火.mp4`
- ✅ 路径空格：`/Users/username/Desktop/My Video.mp4`
- ✅ 特殊字符：`Video (2024) [Final].mp4`

## 🚀 升级方式

### 用户

1. 下载新版本 `.dmg` 文件
2. 拖拽到 `Applications` 文件夹
3. 双击运行即可使用

**注意**：新版本不需要安装 FFmpeg，开箱即用！

### 开发者

```bash

# 运行开发版本

# 构建 Release 版本
```
