### 3. 键盘快捷键系统 ⌨️

**支持的快捷键**:

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `空格` | 播放/暂停 | 切换播放状态 |
| `←` | 快退 5 秒 | 向后跳转 |
| `→` | 快进 5 秒 | 向前跳转 |
| `Shift + ←` | 单帧后退 | 精确控制（约 33ms） |
| `Shift + →` | 单帧前进 | 精确控制（约 33ms） |
| `↑` | 增加音量 | 每次增加 10% |
| `↓` | 降低音量 | 每次降低 10% |
| `R` | 从头播放 | 跳转到视频开头并播放 |

**实现**:
- `KeyboardHandler` 组件：全局键盘事件监听
- `KeyboardShortcutsDialog`：快捷键帮助对话框
- 自动焦点管理：启动时自动请求焦点
- 浮动 SnackBar：快捷键操作时显示临时提示

**快捷键帮助对话框**:
```
┌─────────────────────────────────────┐
│  键盘快捷键                          │
│                                     │
│  [ 空格 ]     播放 / 暂停          │
│  [ ←   ]     快退 5 秒             │
│  [ →   ]     快进 5 秒             │
│  [Shift+←]   单帧后退              │
│  [Shift+→]   单帧前进              │
│  [ ↑   ]     增加音量              │
│  [ ↓   ]     降低音量              │
│  [ R   ]     从头开始播放          │
│                                     │
│                    [关闭]          │
└─────────────────────────────────────┘
```

## 📊 技术实现

### 键盘事件处理

**组件结构**:
```
KeyboardHandler (FocusScope)
├── FocusNode (管理焦点)
├── KeyboardListener (监听键盘事件)
└── child (整个应用)
```

**关键代码**:
```dart
KeyboardListener(
  focusNode: _focusNode,
  autofocus: true,
  onKeyEvent: _handleKeyEvent,
  child: widget.child,
)
```

**事件处理**:
```dart
void _handleKeyEvent(KeyEvent event) {
  if (event is KeyDownEvent) {
    switch (event.logicalKey) {
      case LogicalKeyboardKey.space:
        provider.togglePlayPause();
        break;
      // ... 其他快捷键
    }
  }
}
```

### 音波可视化集成

**位置**: 视频播放器底部（仅在播放时显示）

**布局**:
```dart
Column(
  children: [
    // 音波可视化
    AudioWaveform(
      isPlaying: provider.isPlaying,
      progress: currentPosition / totalDuration,
    ),
    // 进度条
    VideoProgressIndicator(...),
  ],
)
```

## 📝 使用说明

### 键盘快捷键

**基本操作**:
1. 加载视频后自动获得焦点
2. 按 `空格` 开始播放
3. 使用 `←` `→` 快速跳转
4. 按 `R` 重新播放

**精确控制**:
- 按住 `Shift` + 方向键进行单帧控制
- 适用于精确编辑场景

**音量控制**:
- 使用 `↑` `↓` 调节音量
- 当前为占位实现（video_player 不支持音量控制）

### 音波可视化

**自动显示**:
- 播放时自动出现在进度条上方
- 暂停后消失，保持界面简洁

**颜色含义**:
- 主题色 = 已播放
- 红色 = 当前进度位置
- 半透明 = 未播放
