# 🎮 ExtractAudio v2.1 - 快速参考指南

## ⌨️ 键盘快捷键

| 按键 | 功能 |
|------|------|
| **空格** | 播放/暂停 |
| **→** | 快进 5 秒 |
| **←** | 快退 5 秒 |
| **Shift + →** | 单帧前进 |
| **Shift + ←** | 单帧后退 |
| **↑** | 增加音量 (+10%) |
| **↓** | 降低音量 (-10%) |
| **R** | 从头播放 |

**提示**: 点击标题栏的 ❓ 按钮查看完整快捷键列表

---

## 🎵 实时音波可视化

### 显示位置
- 播放时自动出现在视频底部
- 位于进度条上方

### 颜色含义
- 🔵 **蓝色高亮** = 已播放部分
- 🔴 **红色竖线** = 当前进度位置
- ⚪ **半透明** = 未播放部分

### 动画效果
- 播放时：波形上下波动（1.5秒循环）
- 暂停时：停止动画

## 🚀 快速上手

### 第一次使用
1. **拖入视频** → 自动加载到播放器
2. **按空格** → 开始播放
3. **观察音波** → 底部显示动态波形
4. **拖动滑块** → 选择要提取的片段
5. **选择音轨** → 勾选要提取的音轨
6. **点击"质量"** → 选择音频质量
7. **点击"开始提取"** → 完成后显示通知

### 键盘控制高手
1. **空格** 播放
2. **→** 快进到想要的位置
3. **Shift+→** 精确调整
4. **拖动滑块** 选择时间范围
5. **开始提取**

---

## 💡 专业技巧

### 精确剪辑
1. 播放视频找到开始位置
2. 按 `Shift+→` 单帧精确到起始帧
3. 拖动**开始滑块**到该位置
4. 同样方式找到结束位置
5. 拖动**结束滑块**
6. 提取精确片段

### 快速浏览
- 连续按 **→** 快速跳转
- 按 **R** 重新播放
- 使用滑块预览不同位置

## 🐛 遇到问题？

### 应用无响应
- 检查是否有视频加载
- 尝试重新拖拽文件

### 键盘快捷键不工作
- 点击应用窗口获得焦点
- 检查是否有其他应用占用焦点

### 音波不显示
- 确保视频正在播放
- 音波仅在播放时显示
