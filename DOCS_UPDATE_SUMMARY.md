## 🎯 版本 2.5.1 关键信息

### 主要修复
1. 视频解析失败 → 使用 jsonEncode()
2. 音频提取失败 → 使用 executeWithArguments()
3. 中文文件名 → FFmpeg Kit 自动处理
4. 路径初始化 → 正确展开 ~ 路径

### 新增功能
1. 拖拽替换提示 → 确认对话框
2. 点击选择视频 → 文件选择器

### 架构升级
- ffmpeg_kit_flutter → ffmpeg_kit_flutter_new 2.0.0
- 外部依赖 → 内置 FFmpeg 库
- 应用大小 → 99MB Universal Binary
