### 4. 音频质量移到标题栏下拉 ✅

**改进**: 创建 `QualitySelector` 组件，集成到 AppBar

**新组件**: `lib/ui/widgets/quality_selector.dart`

**功能**:
- 紧凑的下拉菜单（图标 + 简短标签）
- 三个质量预设：
  - ⭐ 高质量 (320 kbps)
  - ✓ 标准 (192 kbps)
  - 🗜 压缩 (128 kbps)
- 当前选中项显示勾号标记

**AppBar 布局**:
```
┌──────────────────────────────────────────────────┐
│ ExtractAudio                    [质量▼] [📁输出▼] │
└──────────────────────────────────────────────────┘
```

### 5. 提取完成通知改为临时显示 ✅

**改进**: 使用 SnackBar 代替常驻按钮

**实现**:
- 提取完成后显示绿色 SnackBar
- 包含"打开"按钮和操作
- 5秒后自动消失
- 点击"打开"可查看输出目录

**代码**: `lib/ui/pages/home_page.dart`

**效果**:
```
┌─────────────────────────────────────────────────┐
│ ✓ 提取完成！                    [打开] [打开]   │
└─────────────────────────────────────────────────┘
    ↑ SnackBar（绿色，5秒后消失）
```

---

### 6. 标题栏显示视频标题 ✅

**改进**: AppBar 动态显示视频文件名

**实现**:
- 使用 `Consumer2<ExtractionProvider, VideoPlayerProvider>`
- 当有视频时显示文件名
- 无视频时显示默认标题 "ExtractAudio"
- 文件名过长时自动省略（ellipsis）

**代码**: `lib/ui/pages/home_page.dart`

**效果**:
```
无视频:  "ExtractAudio"
有视频:  "my_concert_video.mp4"
         "very_long_video_name_that_is..." (省略)
```

---

### 7. 添加高级设置面板 ✅

**新组件**: `lib/ui/widgets/advanced_settings_panel.dart`

**功能**:
1. **可折叠面板** (ExpansionTile)
   - 默认折叠状态
   - 点击展开/收起
   - 图标 + 标题 + 副标题

2. **自定义 FFmpeg 参数输入**
   - TextField 输入框
   - 支持任意 FFmpeg 参数
   - 实时预览命令

3. **常用预设快捷按钮**
   - VBR 高质量: `-qscale:a 0`
   - VBR 标准: `-qscale:a 2`
   - 保持原采样率: `-ar 48000`
   - 双声道: `-ac 2`

4. **命令预览**
   - 灰色背景区域
   - 显示完整 FFmpeg 命令
   - 等宽字体显示

5. **重置功能**
   - "重置为默认"按钮
   - 清空自定义参数
   - 显示确认提示

**UI 设计**:
```
┌─────────────────────────────────────────────┐
│ ⚙️ 高级设置              自定义FFmpeg参数 ▼ │
├─────────────────────────────────────────────┤
│ ℹ️ FFmpeg参数说明                            │
│ 这些参数将直接传递给FFmpeg...                │
│                                             │
│ 自定义参数: [________________________]      │
│                                             │
│ 常用预设:                                   │
│ [VBR高质量] [VBR标准] [保持原采样率] [双声道]│
│                                             │
│ 命令预览:                                   │
│ ffmpeg -qscale:a 0 -ar 48000               │
│                                             │
│                                   [重置为默认]│
└─────────────────────────────────────────────┘
```
